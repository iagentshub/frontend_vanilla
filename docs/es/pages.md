# Páginas

La plataforma está organizada en secciones accesibles desde la navegación principal.

---

| Sección | Descripción |
|---|---|
| **Chat** | Interfaz principal. Abre una conversación con cualquier agente configurado. |
| **Agentes** | Crear, configurar y gestionar agentes. Definir sus instrucciones y asignarles skills. Incluye un botón de lupa (esquina superior derecha) que abre el catálogo de agentes públicos: se puede explorar y copiar cualquier agente al listado privado con "Fork" para personalizarlo. |
| **Conexiones** | Añadir y gestionar las claves de API de los proveedores de IA. Cada tarjeta muestra el total de tokens consumidos a través de esa conexión; al pasar el cursor aparece el desglose entre tokens de entrada y de salida. |
| **Memoria** | Consultar y editar la memoria persistente de cada agente. |
| **Skills** | Explorar las skills disponibles por idioma y categoría. |
| **Perfil** | Ajustes del usuario: tema visual y contraseña. |
| **Administración** | Gestión de cuentas de usuario. Solo accesible para administradores. |
